* EXCEPTION:
~SevenSegnment~ does not look like a rendering primitive.

The object must adhere to AT LEAST ONE of the following be considered a primitive:
- Object must respond to ~primitive_marker~ returning either ~:solid~, ~:label~, ~:sprite~, ~:line~, or ~:border~.
- Object must respond to ~:x~, ~:y~, ~:w~, ~:h~, and ~:path~.
- Object must respond to ~:x~, ~:y~, and ~:text~.
- Object must implement ~draw_override~.
